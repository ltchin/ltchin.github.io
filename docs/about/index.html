About This Site

This site is a hybrid of Jonathan McGlone’s website in Jekyll and Yihui Xie’s XMin template for Hugo. I ported most of my old Bootstrap + Jekyll website to a version where I try to

  1. understand every line of code rather than copy-pasting wholesale
  2. minimize Javascript usage
  3. make the website accessible to screen readers
  4. make the website accessible to low bandwidth users

Please let me know if you have some recommendations for how to better achieve those goals. I particularly welcome feedback from users with disabilities.
